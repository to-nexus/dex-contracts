## 🎯 DEX V2란 무엇인가?

DEX V2는 오더북을 통해 토큰 거래를 가능하게 하는 탈중앙화 거래소 프로토콜입니다. 자동화된 마켓 메이커(AMM)와 달리, 이 DEX는 지정된 가격에서 매수 및 매도 주문을 매칭하는 전통적인 오더북 모델을 사용합니다.

### 주요 특징

- **오더북 거래**: 매수 및 매도 주문이 오더북을 통해 매칭되어 거래 가격을 직접 제어할 수 있습니다
- **유연한 수수료 구조**: 각 마켓은 maker와 taker에 대해 서로 다른 수수료율을 가질 수 있습니다
- **다중 마켓**: 동일한 quote 토큰에 대해 여러 마켓이 존재할 수 있으며, 각 마켓은 다른 수수료 정책을 가질 수 있습니다
- **네이티브 CROSS 지원**: 수동 래핑 없이 Cross Chain의 네이티브 CROSS 코인을 직접 사용할 수 있습니다

## ✨ 주요 기능

### 주문 유형

#### 지정가 주문

원하는 가격에 주문을 등록할 수 있습니다. 주문은 다른 사람이 지정한 가격에 매칭될 때까지 오더북에 대기합니다.

**시간 제약 옵션:**
- **Good Till Cancel (GTC)**: 주문이 체결되거나 취소될 때까지 활성 상태를 유지합니다
- **Immediate Or Cancel (IOC)**: 주문이 지정 가격 또는 더 나은 가격에 즉시 체결되며, 미체결 부분은 취소됩니다
- **Fill Or Kill (FOK)**: 주문이 지정 가격 또는 더 나은 가격에 완전히 체결되어야 하며, 그렇지 않으면 취소됩니다

#### 시장가 주문

오더북에서 가능한 최상의 가격으로 주문을 즉시 실행합니다. 대기 없이 바로 거래하고 싶을 때 사용하기 좋습니다.

### 네이티브 CROSS 코인 지원

DEX는 Cross Chain의 네이티브 CROSS 코인을 원활하게 지원합니다:

- **CROSS 직접 전송**: 트랜잭션과 함께 네이티브 CROSS 코인을 직접 전송할 수 있습니다 (수동 래핑 불필요)
- **자동 처리**: 시스템이 거래를 위해 CROSS를 자동으로 래핑하고 지갑으로 전송할 때 자동으로 언래핑합니다
- **투명한 경험**: 추가 단계 없이 CROSS를 다른 토큰처럼 사용할 수 있습니다

## 💰 수수료

DEX V2는 네 가지 별도의 수수료율을 사용하는 유연한 수수료 구조를 사용합니다:

1. **판매자 Maker 수수료**: 지정가 주문을 등록하는 판매자의 수수료
2. **판매자 Taker 수수료**: 시장가 주문을 실행하는 판매자의 수수료
3. **구매자 Maker 수수료**: 지정가 주문을 등록하는 구매자의 수수료
4. **구매자 Taker 수수료**: 시장가 주문을 실행하는 구매자의 수수료

### 수수료 작동 방식

- 각 마켓은 자체 수수료율을 설정합니다
- **Maker 주문** (지정가 주문): 오더북에 유동성을 제공하므로 일반적으로 수수료가 낮습니다
- **Taker 주문** (시장가 주문): 유동성을 소비하므로 일반적으로 수수료가 높습니다
- 매수와 매도 주문에 대해 수수료가 다를 수 있습니다

이 수수료 구조를 통해 마켓은 지정가 주문을 장려하기 위해 낮은 maker 수수료를 제공하는 것과 같은 특정 거래 행동을 유도할 수 있습니다.
